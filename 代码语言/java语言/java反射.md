# java   reflect  （反射）









```

加载完类之后，在堆内存中，就产生了ー个 Class类型的对象（一个
类只有一个 Class对象），这个对象就包含了完整的类的结构信息
我们可以通过这个对象看到类的结构。这个对象就像一面镜子，透过
这个镜子看到类的结构，所以，我们形象的称之为：反射。


每一个类被加载到内存中的时候，都会创建一个Class对象，每个类只有一个Class对象，这个Class对象中包含了关于这个类的全部信息，我们可以通过Class对象 创建这个类的实例，调用这个类实例对象的方法等等
Class对象就像一面镜子，将类的结构(属性，方法)全部反映了出来，透过这个镜子看到类的结构，所以，我们形象的称之为：reflect,

reflect有下列的翻译
(反映; 映出(影像); 反射(声、光、热等); 显示，表明，表达(事物的自然属性或人们的态度、情感等);)

好多地方都将reflect翻译成反射，其实翻译成反映，显示更好吧


```
